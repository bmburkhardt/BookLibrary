# BookLibrary
(Project for school. Created by myself and Bryce Boeson)

- Book inventory system to track books, authors, publishers, and author payout amounts. The system also keeps an audit trail for any changes made to a record. Program also allows for report generation based on the userâ€™s role. The system is used through a fully functional GUI, created using JavaFX.
